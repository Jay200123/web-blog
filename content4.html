Operating Systems

What is an operating system?

The Operating System is the most essential part of software that runs on a computer. It regulates the memory, storage, software, and hardware of the computer. A computer cannot function and perform without an operating system.

The Operating System's job

The Operating System is an essential portion of a computer that controls both the hardware and software of a computer. Most of the time, computer programs are all running once, and the entire programs require access to the CPU, RAM and Storage of your computer. To guarantee that every software receives the resources it needed to run, the operating systems managed all of this.

Types of Operating Systems

Every Computer that is being purchased, comes with a preinstall Operating System. The default Operating system that comes with computers is accustomed by the majority of users, yet it is possible to update or change operating systems. Microsoft Windows, MacOS and Linux are the most popular Operating systems for private computers. Present Day operating systems use a(GUI) Graphical User Interface, that lets the user use the mouse to navigate to the computer like for example clicking icons, running applications etc. of the computer's software, specifications are displayed on the screen with the mixture of graphics and text.

Microsoft Windows

Windows OS was introduced in the 1980s by the Microsoft Corporation. There have been various versions of Windows, but the most current ones are Windows 10 released in 2015, Windows 8 relesed in the year 2012, Windows 7 released in the year 2009, and Windows Vista released in the year 2007. Windows are pre-installed on most new PCs, which makes it the most popular operating system in the world.

macOS

The MacOS (earlier known as OS X) is a range of Operating Systems introduced by Apple Corporation. It was pre-installed on all Macintosh PC known as Macs. Certain versions include the Sierra (released in 2016), High Sierra(released in 2017) and Mojave (released in 2018). Based on StatCounter Global Stats, accounts of users from MacOS are only 10% of the global operating systems which is much lower than Windows users which contribute more than 80%. The reason for these lower rates is Apple Computers tend to be more expensive, yet some people prefer the MacOS over Windows OS.

Linux

The Linux family of open source operating systems, which implies that they can be modified and distributed by anyone around the world. LinuxOS is distinctive from private software like Windows, which can only be modified by the Microsoft that owns the windows. The edge of Linux are it is free, and there are many diverse distributions or versions you can freely choose from. As per StatCounter Global Stats, Linux users account for only less than 2% of global operating systems. Still, most servers run LinuxOS since it's somewhat easy to customize.
